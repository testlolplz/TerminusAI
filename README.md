# 🤖 TerminusAI

**TerminusAI** is a sophisticated AI assistant designed exclusively for Termux, bringing the power of advanced Large Language Models directly to your Android terminal. Completely free and open-source — no credit card, no subscriptions, no limitations.

---

## ✨ Key Features

### 🧠 Intelligence & Chat
- **Multi-Model Support**: Access Gemini, Llama, Qwen, and DeepSeek through OpenRouter's free tier
- **Persistent Conversations**: Save, load, and review your chat history
- **Custom AI Personalities**: Create tailored prompts to customize AI behavior

### 💻 Development Tools
- **Code Execution**: Run Python scripts directly within the terminal
- **Syntax Highlighting**: Beautiful code formatting for readability
- **Code Analysis**: Get intelligent feedback on your code

### 🌐 Web & Information
- **Web Search**: Query the internet for real-time information
- **News Updates**: Stay informed with latest news feeds
- **Weather**: Get current weather and forecasts
- **QR Code Generation**: Create QR codes on the fly

### 📂 File Management
- **Read & Write Files**: Seamlessly work with text and data files
- **File Downloads**: Fetch and save files from the web
- **File Organization**: Complete file system navigation

### 🔊 Accessibility
- **Text-to-Speech**: Convert responses to audio output
- **Voice Integration**: Accessible AI assistance

### ⚙️ Productivity & Utilities
- **Todo Lists**: Manage tasks and reminders
- **Password Generator**: Create secure passwords
- **System Information**: Monitor device stats and storage
- **Backup System**: Export chats to markdown and backup data

### 🎮 Advanced Control
- **25+ Slash Commands**: Comprehensive command suite for advanced users
- **Quick Actions**: Efficient workflows with intuitive commands

---

## 📋 Requirements

Before getting started, ensure you have:

- **Termux** (install from [F-Droid](https://f-droid.org/), **NOT** Google Play)
- **Python 3.7 or higher**
- **Active internet connection**
- **Free OpenRouter API key** (get one at [openrouter.ai](https://openrouter.ai))

---

## 🚀 Installation

### Quick Start (One Command)

```bash
curl -s https://raw.githubusercontent.com/testlolplz/TerminusAI/main/install.sh | bash
```

This command will:
1. Download the installer script
2. Install all dependencies
3. Configure TerminusAI for your system
4. Prompt for your OpenRouter API key

### Manual Installation

For detailed installation steps or troubleshooting, see the [install.sh](./install.sh) script.

---

## 🔑 Getting Your Free API Key

1. Visit [openrouter.ai](https://openrouter.ai)
2. Sign up for a free account
3. Generate an API key from your dashboard
4. Paste it when prompted during installation

## 💡 Usage Tips

- **Commands**: Type `/help` in the chat to see all available commands
