# 🤖 AI Coding Assistant for Acode

[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com/yourusername/acode-ai-assistant)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![Acode](https://img.shields.io/badge/Acode-1.0.0%2B-orange.svg)](https://acode.foxdebug.com/)

An intelligent AI-powered coding assistant plugin for Acode editor on Android, bringing ChatGPT-like capabilities directly into your mobile coding environment.

## ✨ Features

### 🤖 AI Chat Assistant
- Interactive chat with AI about your code
- Context-aware responses based on current file
- Code insertion directly from chat
- Conversation history with context retention

### ⚡ Smart Code Completion
- Automatic code suggestions as you type
- Context-aware completions based on your codebase
- Support for 15+ programming languages
- Intelligent function and variable naming

### 🛠️ One-Click Actions
- **Explain Code** - Get detailed explanations with examples
- **Debug Code** - Find and fix bugs automatically
- **Optimize Code** - Improve performance and readability
- **Generate Documentation** - Auto-create JSDoc, docstrings, etc.
- **Convert Language** - Translate between programming languages
- **Generate Tests** - Create unit tests (Jest, pytest, JUnit)
- **Code Review** - Professional code quality analysis
- **Security Check** - Vulnerability scanning (SQLi, XSS, etc.)

### 🎨 Quick Actions Menu
- Floating button for instant access
- 15+ one-click operations
- Context-aware actions based on selection
- Customizable menu items

### ⌨️ Keyboard Shortcuts
| Shortcut | Action |
|----------|--------|
| `Ctrl + Shift + A` | Open AI Assistant |
| `Ctrl + Shift + E` | Explain selected code |
| `Ctrl + Shift + D` | Debug selected code |
| `Ctrl + Shift + O` | Optimize code |
| `Ctrl + Shift + G` | Generate documentation |
| `Ctrl + Shift + T` | Generate tests |

### 🔧 Custom Commands
Create your own AI prompts with custom triggers:
```json
{
    "trigger": "review",
    "prompt": "Review this code for best practices:"
}
